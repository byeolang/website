Detailed Description

User-defined class origin in byeol language.

Represents user defined classes in managed codes. Origin should be shadowed to 'obj' type. Every interaction with origin should be handled through obj class. Direct inheritance is not allowed.

Note
Use tstr<obj> instead of tstr<origin> for binders to avoid unexpected behavior.
