template<typename T>
class by::tbaseObjOrigin< T >

Template for easily defining baseObj origin objects

A class template used to easily define origin objects for baseObj.

Remarks
See origin and baseObj first.

The structure and concept are identical to the origin class. It shadows its own type and is intended to be treated as baseObj type. However, differences arise in usage. This is because the baseObj class concept differs from obj.

Remarks
Unlike obj, baseObj's origin object exists only once. See baseObj for details.
경고
This limitation affects binder usage too. Simply declaring a binder with type parameter 'baseObjOrigin' is not allowed. Use 'baseObj' type instead.
tstr<baseObjOrigin> a; // X, unexpected behavior may happen.
tstr<baseObj> a; // O
