Detailed Description

Error report collector.

A class that holds baseErr objects. When there's logic that takes a long time or performs complex operations, function depth becomes very deep. When errors occur in inner functions, use errReport to record errors and continue work as much as possible while recording that errors occurred.

Container

Since errReport's almost all functionality is managing baseErr objects, it has an interface similar to tucontainable.

Noisy

Some errReports want to automatically perform log() each time a new err object is added. In this case, specify setNoisy(true).
