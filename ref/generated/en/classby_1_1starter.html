Detailed Description

Program execution starter.

A kind of worker that executes AST verified by verifier. If an exception occurs in is required to run an autopod during execution, it dumps logs and returns the exception. See worker for more details if unfamiliar with the worker concept.

Main Function Look Up

The main function must meet the following conditions:

  1. Found directly under root (pod)
  2. Name must be main
  3. Return type must be void or int
  4. Main function must have 1 or more statements

Usage Example

Can be used with interpreter to parse and execute byeol code.

interpreter ip;
ip.work();
if(!ip.isVerified()) return -1;
str res = starter().setTask(ip.getSubPod()).work();
if(res)
if(res->isSub<nInt>()) return *res->cast<nInt>();
return -1;
