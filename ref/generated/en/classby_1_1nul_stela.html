Detailed Description

Null stela node representing empty or undefined values.

When retrieving child objects from Stela, usually by name, if no child with that name exists, nulStela is returned instead.

nulStela implements the null object pattern. When value conversion is requested on this object, it always returns the default value for that type.

Remarks
Checking for nulStela To verify if a stela object is nulStela, check if isExist() or operator bool() returns false.
