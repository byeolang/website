Detailed Description

Value stela node for storing primitive values.

Unlike nulStela, this Stela holds scalar types like int, float, string. Internally stores value as a base string, and can convert to appropriate types on type conversion requests like asInt().

Remarks
Exception on type conversion failure If type conversion fails, an exception is thrown, so be careful.
